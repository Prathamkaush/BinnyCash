# 🪐 BinnyCash — Earn Real Money & Rewards Online

BinnyCash is a premium, modern, and fully localized micro-earning web application. Users can earn real cash and rewards by testing apps, playing games, completing digital offers, and taking market research surveys. With global multi-currency withdrawals and high-converting referral commissions, BinnyCash enables users to monetize their spare time worldwide.

The platform is designed with rich aesthetics, sleek animations, a dark-violet cyberpunk gradient theme, and full translation support for **21 languages**.

---

## ⚡ Key Features

*   **Interactive Earning Dashboard:** An immersive space-themed user dashboard equipped with an active task list, mock task completion engine with real-time balance updates, payout wallet interface, and a referral copy utility.
*   **Live Cashouts Feed:** A real-time marquee banner displaying the latest withdrawals made by active users globally (e.g., `pratham***` withdrew `$15.40` via PayPal). The ticker simulates new payouts every 6 seconds.
*   **Comprehensive Localization (21 Languages):** Dynamic client-side language switching supported by standard JSON-like dictionaries in TypeScript, storing user preferences in `localStorage`.
*   **Modern Interactive Aesthetics:** Incorporates a dynamic twinkling starfield background, floating rotating shapes (glowing green outlines of stars, hexagons, and squares), smooth hover states, custom transitions, and a scroll-triggered sticky call-to-action (CTA) widget.
*   **Multiple Reward Verticals:** Highlighting paid surveys, mobile game milestones (Play-to-Earn up to $60.00/game), and quick app installs/offers.
*   **Robust Payout Gateways:** Withdrawal forms that validate account addresses and minimum limits ($2.00 minimum for Litecoin, $5.00 minimum for PayPal and digital gift cards).
*   **Lifetime Referrals:** Referral system rewarding users with a lifetime 10% commission on all earnings generated by friends invited through their unique referral link.

## 🛠️ Technology Stack

*   **Framework:** [Next.js 16.2.9](https://nextjs.org/) (App Router layout, Client Components).
*   **Library:** [React 19.2.4](https://react.dev/) (Hooks: `useState`, `useEffect`, `useRef`).
*   **Styling & Design System:** [Tailwind CSS v4](https://tailwindcss.com/) with PostCSS configuration.
*   **Icons:** [Lucide React](https://lucide.dev/) for crisp, responsive vectors.
*   **Fonts:** Google Fonts via `next/font` — **Inter** (sans-serif utility font) & **Outfit** (modern display font).
*   **Deployment & Performance:** Built to deploy seamlessly on platforms like [Vercel](https://vercel.com).
*   **Assets:** Dynamic flag icons sourced from [FlagCDN](https://flagcdn.com/) for localized select options.

## 🧩 Architectural Highlights

### 1. State Router & Persistent Language Selector
In [src/app/page.tsx](file:///c:/Users/Pratham%20Kaushik/Desktop/extra/BinnyCash/src/app/page.tsx), a simplified view state router handles transition states between the `"landing"`, `"about"`, and `"dashboard"` interfaces:
```typescript
const [currentView, setView] = useState<string>("landing");
const [language, setLanguage] = useState<string>("en");
```
When a language is switched in [src/components/Header.tsx](file:///c:/Users/Pratham%20Kaushik/Desktop/extra/BinnyCash/src/components/Header.tsx), the choice is saved to `localStorage` as `binny-lang`, making sure user preference is remembered across refreshes.

### 2. Interactive Task Dashboard
Located in [src/components/Dashboard.tsx](file:///c:/Users/Pratham%20Kaushik/Desktop/extra/BinnyCash/src/components/Dashboard.tsx), this component simulates a real task dashboard:
*   **Profile:** Embellished with a custom SVG **Space Bunny Astronaut** avatar.
*   **Balance Wallet:** Updates dynamically. Clicking "Complete" on any offer invokes a loading timer, simulating client-server verification before crediting the mock cash reward.
*   **Withdrawal System:** Checks for wallet limits and shows customized validation errors for PayPal, Crypto, and Gift Cards.

### 3. Live Marquee Ticker
Located in [src/components/LiveFeed.tsx](file:///c:/Users/Pratham%20Kaushik/Desktop/extra/BinnyCash/src/components/LiveFeed.tsx), the component loops cashout listings using a CSS marquee:
```css
@keyframes ticker {
  0% { transform: translateX(0); }
  100% { transform: translateX(-50%); }
}
```
An interval runs every 6 seconds to inject random user withdrawals, keeping the feed alive.

## 🚀 Getting Started

To run the project locally on your machine, follow these simple steps:

### 1. Prerequisites
Ensure you have [Node.js](https://nodejs.org/) installed (v18.x or higher is recommended) along with standard package managers.

### 2. Install Dependencies
Navigate to the root directory of `BinnyCash` and run:
```bash
npm install
```

### 3. Run Development Server
Start the local server with:
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) on your web browser to view the application.

### 4. Build Production Bundle
To create a production-optimized build, run:
```bash
npm run build
```
Start the production server locally to test performance:
```bash
npm run start
```

### 5. Code Quality Linting
To check and report any ESLint issues:
```bash
npm run lint
```
